How it works
Each floor cell starts with a dirt level (1-3) and needs that many robot visits to become fully clean. The robot has a limited battery; once it drops to the low-battery threshold it path-finds (BFS) to the nearest charging station, recharges, then resumes. Zigzag blindly re-sweeps the whole room in a fixed pattern three times. Smart always BFS-searches for the nearest still-dirty cell, so it never wastes moves revisiting clean ground.
